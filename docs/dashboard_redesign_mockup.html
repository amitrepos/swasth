# Element Current Position Proposed Position Size Change Why
1 Health Score Ring (Heart) #1 #1 (unchanged) Same Emotional anchor for elderly patients. Stays.
2 Vitals (BP + Sugar) #6 #2 ▲ Same Most important daily data. Should be immediately visible.
3 Today's Meals #5 (tiny badge) #3 ▲ 2x bigger Used 2-3x daily. Added meal slot prompts (Breakfast/Lunch/Dinner).
4 AI Insight #3 (half viewport) #4 ▼ ~60% smaller Collapsed to 2 lines + "Read more". Stops dominating the screen.
5 Physician Card #4 #5 Same Slightly lower but still prominent. Doctor connection matters.
6 BMI Bar #7 (full-width) Merged into Physician ~75% smaller Changes 1x/month. Demoted to inline badge under physician card.
7 Vital Summary (90-day) #2 #6 ▼ Same Trends are "check weekly" not "check daily". Pushed below daily actions.
8 Quick Actions #8 #7 Same Footer utility actions stay at bottom.
9 Meal badge icon ❗ (judgmental) 🍜 (food emoji) Icon swap Per Dr. Rajesh + Healthify: exclamation is shaming. Use food emoji.

Key Design Principles

  • Daily actions visible first — BP/Sugar readings + Meals are above the fold
  • Heart stays — emotional anchor for 65yo patients in Bihar
  • AI insight doesn't dominate — 2-line preview, expand on tap
  • BMI shrinks — static metric doesn't deserve same space as daily vitals
  • Meal slots guide behavior — empty slots prompt "Tap to log" for each meal
  • No shaming icons — food emojis instead of exclamation marks
